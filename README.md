# Instrucciones 

1. **Crear entorno:** python -m venv _env
2. **Activar entorno**: 
    **Windows:** .\_env\Scripts\activate
    **Lunix o Mac:** source _env/bin/activate
3. **Instalar dependendias o librerias**: pip install -r requirements.txt
